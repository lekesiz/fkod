## 5. Teknoloji Stack Özeti

### Frontend
- **Framework:** React 19 + React Native
- **Build:** Vite
- **Styling:** Tailwind CSS 4
- **State:** Redux Toolkit + Zustand
- **Navigation:** React Navigation + Wouter
- **UI Components:** shadcn/ui + React Native Paper

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** PostgreSQL 15
- **Cache:** Redis (Upstash)
- **ORM:** Raw SQL (node-postgres)

### DevOps
- **Frontend Hosting:** Vercel
- **Backend Hosting:** Heroku
- **Database:** AWS RDS
- **CDN:** Vercel + CloudFront
- **Monitoring:** Sentry + DataDog
- **Logging:** Winston + ELK Stack

### Mobile
- **Framework:** React Native + Expo
- **State:** Redux Toolkit
- **Storage:** AsyncStorage + SQLite
- **Notifications:** Expo + Firebase
